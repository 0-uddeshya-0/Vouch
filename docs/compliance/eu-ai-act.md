# EU AI Act Compliance

## Risk Classification

Vouch is classified as **Limited Risk** under the EU AI Act (Article 52 - Transparency obligations).

## Requirements

### Transparency Obligations

1. **AI System Disclosure**
   - Clear identification in all PR comments
   - Model information (Haiku/Sonnet)
   - Confidence scores

2. **Human Oversight**
   - One-click dismissal mechanism
   - Override via PR labels
   - Dashboard for bulk management

3. **Audit Trail**
   - All decisions logged
   - Exportable to CSV/JSON
   - 7-year retention (BSI Grundschutz)

### Technical Documentation

Required for conformity assessment:

1. System architecture
2. Model specifications
3. Data processing policies
4. Security measures
5. Accuracy metrics
6. Known limitations
